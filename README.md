# 📚 Exam Flashcards — Professional English

Interactive flashcard app for exam prep on the subject **Professional Foreign Language (IT topics)**.

## What's inside

20 flashcards covering all topics from the exam list:

- Digital divide in Kazakhstan
- Cybercrime (types, effects, prevention)
- Safe internet use & online self-protection
- Password safety, social media threats
- Data storage devices
- Emerging IT trends
- Network issues

All answers are written at **Elementary English level** — short sentences, simple vocabulary.

## How to use

Open `flashcards.html` in any browser. No installation needed.

| Action | Control |
|--------|---------|
| Flip card (see answer) | Click the card / `Space` |
| Next card | `→` arrow key or **Next** button |
| Previous card | `←` arrow key or **Prev** button |
| Shuffle order | **🔀 Shuffle** button |
| See all Q&A at once | **Show all cards as list** at the bottom |
